Payments Service — v2 Spec

Draft · last edited just now

Overview

The payments service handles all transaction processing for checkout. It should be fast and reliable.

Goals

  • Process payments quickly
  • Support cards, wallets, and bank transfers
  • Be very secure

API

POST /v2/charge creates a new charge. It returns a charge object.

Open Questions
